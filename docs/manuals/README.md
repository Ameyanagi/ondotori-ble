The continuously maintained [RTR500B Series Help](https://tandd.com/support/webhelp/rtr500b/eng/)
explains model behavior and Bluetooth state, while T&D's complete
[manual index](https://tandd.com/manual/) includes base-unit manuals.

The XML specification confirms T&D's little-endian two-byte temperature and
humidity conversion, significant digits, and error marker. It does not publish
the manufacturer-advertisement envelope or serial-family mapping.
The separate [communication specification](https://www.tandd.co.jp/techinfo/)
requires agreeing to T&D's terms and applying directly. Those terms prohibit
passing the received specification to another person, so it has not been
downloaded or copied into this project.
